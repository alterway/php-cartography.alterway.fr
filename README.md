# PHP Cartography

This tool generate a tree of tools, based on the [ziadoz list](https://github.com/ziadoz/awesome-php).

## Usage

+ add a vhost on the `web` directory.
+ configure cron to run `update.php` every day (example: `0 0 0	* *	*	/path/update.php`)
